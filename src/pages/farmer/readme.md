
EcoCold — Solar Powered Cold Storage Platform
EcoCold is a React-based web application that allows farmers and cold storage operators to manage solar-powered refrigeration hubs. The platform helps reduce post-harvest losses by enabling easy storage booking and monitoring.
🌱 Project Purpose
Farmers often lose produce due to lack of refrigeration. EcoCold provides a digital platform to discover, book, and monitor solar-powered cold storage facilities.
🚀 Features
User authentication (Farmer / Operator)
Storage hub discovery
Booking and payment flow
AI temperature monitoring alerts
Farmer dashboard
Operator monitoring dashboard
Responsive design
Interactive temperature charts
